# Исследование возможности прогнозирования состояния реакторной установки ВВЭР-1000 по показаниям СВРК с применением методов машинного обучения

## Описание проекта

Данный проект реализует полный цикл анализа данных системы внутриреакторного контроля (СВРК) реакторной установки ВВЭР-1000 с применением методов машинного обучения для прогнозирования состояния реактора.

## Методика обработки и анализа данных

### Этап 1: Предобработка данных

1. **Удаление константных признаков** - исключение параметров с нулевой дисперсией или почти константных значений
2. **Интерполяция пропусков** - линейная интерполяция отсутствующих значений
3. **Сглаживание скользящим средним** - уменьшение шумов измерений (окно = 5 точек)
4. **Нормализация** - стандартизация данных (StandardScaler: μ=0, σ=1)

### Этап 2: Кластеризация временных точек

Метод **k-средних (K-Means)** применяется для выделения устойчивых режимов работы реактора:

- Оптимальное число кластеров определяется по **силуэтному коэффициенту**
- Для данного набора данных: **k=4** кластера
- Выделенные режимы:
  - **Кластер 0**: Стационарная работа на мощности (~98% времени)
  - **Кластер 1**: Переходные процессы разогрева/охлаждения
  - **Кластер 2**: Режимы сниженной мощности (ночной провал)
  - **Кластер 3**: Промежуточные переходные режимы

### Этап 3: Метод главных компонент (PCA)

Для каждого кластера отдельно применяется PCA:

- **Цель**: снижение размерности при сохранении 95% дисперсии
- **Результат**: выявление специфических корреляционных структур для каждого режима
- **Визуализация**: проекции на первые две главные компоненты подтверждают компактность кластеров

### Этап 4: Прогнозирование LSTM

Архитектура нейронной сети для прогноза тепловой мощности реактора:

- **Входные данные**: главные компоненты PCA
- **Окно истории**: 10 часов
- **Горизонт прогноза**: 1 час
- **Архитектура**:
  - LSTM слой (64 единицы, ReLU)
  - Dropout (0.2)
  - Dense слой (32 единицы, ReLU)
  - Dropout (0.1)
  - Выходной Dense слой
- **Обучение**: 80% данных - train, 20% - валидация
- **Функция потерь**: MSE
- **Оптимизатор**: Adam (lr=0.001)

## Параметры запуска

Основные параметры можно изменить в блоке `if __name__ == "__main__"`:

```python
DATA_PATH = '/workspace/Пример данных.xlsx'     # Путь к данным измерений
PARAMS_PATH = '/workspace/Список параметров.xlsx'  # Путь к описанию параметров
SAVE_DIR = '/workspace/SVRK'                    # Директория для результатов

results = run_full_pipeline(
    data_path=DATA_PATH,
    params_path=PARAMS_PATH,
    save_dir=SAVE_DIR,
    history_window=10,        # Окно истории (часов)
    forecast_horizon=1,       # Горизонт прогноза (часов)
    variance_threshold=0.95   # Порог дисперсии для PCA
)
```

## Интерпретация результатов

### Кластеры режимов работы

Анализ средних профилей позволяет интерпретировать кластеры:

- **Высокая мощность** (NAKЗ > 2000 МВт): стационарная работа
- **Низкая мощность** (NAKЗ < 500 МВт): ночной провал, останов
- **Переходные процессы**: изменение температур ТВС, концентрации бора

### Качество прогноза

Метрики для оценки модели:

- **RMSE** - среднеквадратичная ошибка (в единицах целевой переменной)
- **MAPE** - средняя абсолютная процентная ошибка
- **R²** - коэффициент детерминации (доля объяснённой дисперсии)

## Научная новизна

1. **Комбинированный подход**: кластеризация + PCA внутри кластеров + LSTM
2. **Режимно-ориентированный анализ**: учёт специфики различных режимов работы реактора
3. **Сохранение 95% дисперсии**: оптимальное снижение размерности без потери информации
4. **Прогнозирование на основе главных компонент**: использование скрытых взаимосвязей параметров

## Требования к данным

Формат входных данных:
- **Excel файл** (.xlsx)
- **Первый столбец**: временные метки (datetime)
- **Остальные столбцы**: параметры СВРК (температуры, давления, мощности и т.д.)
- **Целевая переменная**: тепловая мощность реактора (NAKЗ или аналог)
